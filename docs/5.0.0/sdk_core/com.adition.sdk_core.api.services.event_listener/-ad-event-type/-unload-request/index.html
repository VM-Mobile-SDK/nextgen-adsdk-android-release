UnloadRequest

data object UnloadRequest : AdEventType

Signals that the renderer wants to be closed.

This method is actively used in AdPlacementType.INTERSTITIAL ads to hide ads from the screen, but it can also be used in AdPlacementType.INLINE ads. When this method is called, the application developer is responsible for closing the ad.
