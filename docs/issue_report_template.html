AdSDK Issue Report

Thank you for your contribution to improving the AdSDK.

When creating an issue report, please provide us with the following information and send it to:
mobilesdk@virtualminds.com

  • Description – required
  • Expected behaviour – required
  • Actual behaviour – required
  • SDK version – required
  • Device – if the issue occurs only on specific devices
  • Android version – if the issue occurs only on specific Android versions
  • Android Studio version – if the issue is more related to development than production
  • Steps to reproduce the issue – required
  • Integration method – required GitHub packages or Nexus integration
  • Attachments – required It can include a crash report, logs, or other useful information.
    Also, we need a demo app to reproduce the error. You can use our template
    as a starter project and edit it to reproduce the error.
