placementType

val placementType: PlacementType

Enum class describing the expected type of placement for the Advertisement.

Advertisement matches the placement received from the server with the placement type you provided. With it we can make sure the placement type from the ad setting and the banner response are the same.

The default value is PlacementType.INLINE. If you want to present an Ad as a interstitial you have to select: PlacementType.INTERSTITIAL.

Be careful, do not use interstitial in a Column or LazyColumn, because it will only work properly in a view which could be shown as fullscreen modal.
