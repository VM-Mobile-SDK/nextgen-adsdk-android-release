Event type.

TAP = user triggered a URL, for example by tapping on an Ad IMPRESSION = the ad has been rendered, irregardless if it has been seen by the user or not. Can be triggered if the add is outside of the viewport VIEWABLE_MRC1 = At least 1% of the Ad has been actually seen by the user VIEWABLE_MRC50 = At least 50% of the ad has been seen by the user VIEWABLE_MRC100 = 100% of the ad has been seen by the user
