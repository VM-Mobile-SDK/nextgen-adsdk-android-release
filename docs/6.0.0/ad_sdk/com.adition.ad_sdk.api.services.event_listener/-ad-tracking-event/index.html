AdTrackingEvent

sealed class AdTrackingEvent : Serializable

A sealed class with all tracking events types that SDK can process.

All tracking events are linked to the advertising configuration on the server. They are processed automatically by the SDK.

Typically, you use this sealed class in AdEventListener to identify tracking events that have completed successfully or with an error.

Learn more about AdEventListener in the our tutorials.
