AdEventListener

interface AdEventListener

An interface of object for tracking and processing Advertisement related events.

The object of this interface is used to pass to Advertisement via AdService.makeAdvertisement.

You can create the object of this interface for:

  • Monitoring events of tapping on an advert.

  • Monitoring events related to advertisement tracking.

  • Monitoring events sent from the renderer.

  • Controlling the repeat of failed events.

Learn more in our tutorials.
